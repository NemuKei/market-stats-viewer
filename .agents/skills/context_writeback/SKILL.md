# Purpose
常設コンテキストへの反映可否を条件判定し、必要時のみ最小差分で反映する。

# When to use / When NOT to use
- When to use: 将来の意思決定に影響する情報を正本（`STATUS.md` / `DECISIONS.md`）へ反映するとき
- When NOT to use: 単発メモ・一時的な会話 / 4条件を満たさない / 保存先を1ファイルに特定できない

# Procedure
1. 反映候補が以下4条件を**すべて**満たすか確認する。
   1. 今後2回以上の再利用見込み
   2. 将来の意思決定に影響
   3. 1〜3行で要点化可能
   4. 保存先を1ファイルに特定可能
2. 1つでも不足 → 反映しない。
3. 満たす場合 → 明示された内容だけを最小差分で更新（推測補完しない）。
4. 同順位の情報源で矛盾あり → `DECISIONS.md` に `D-YYYYMM-xxx` で暫定記録、新しい決定を優先。

# Validation
- 4条件の根拠あり / 更新先が1ファイル / 推測追記なし / 矛盾時は `D-YYYYMM-xxx` 記録済み
